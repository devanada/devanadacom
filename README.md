## Run Locally

Clone the project

```bash
  git clone https://github.com/devanada/devanadacom.git
```

Go to the project directory

```bash
  cd devanadacom
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
